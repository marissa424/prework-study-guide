CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Box shadow adds shadow effects around an elements frame
  • Declorations contain two important components. The css propert we want to apply and the value of the property
  • Linking your style sheet: link rel="stylesheet" href="./assets/style.css"

Git

  • Git: Version control software
  • GitHub: Cloud based service that host respositories

      • Pushing Changes to Git

    1. Make sure you're in the correct branch, run command: cd prework-study-guide
    2. Make sure changes go to branch not main, run command: git checkout -b feature/startercode
    3. Save changes in VScode then start commit to changes, run command: git add -A
    4. Commit Message, run command: git commit -m "enter reason for change here"
    5. Next run command: git pull origin main
    6. Next run command: git push origin feature/starter-code

  • Make sure to always push to your branch and not main
  • Make sure when checking pull request that base: main and compare: feature/starter-code are selected

JavaScript

  • A variable is a named contaner that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script file
  • An array is a single variable that is used to hold a group of data
  • There are several types of loops used in JavaScript
  • for loop uses the predictable pattern of indices to perform a task on all the items in an array

  • A for loop contains 3 important statements
    • First Statement: Determines the starting point we declare x and give it value of 0
    • Second Statement: The condition as long as the number in the variable x is less than the length of the array. The loop will keep running.
    • Third Statemen: Allows the array to iterate over each item. Indices follows a predictable pattern, increase by 1
  • The code x++ is a shortcut for x+1
  • A functions name is followed by()
  • Code to be executed is written inside curly brackets
  • Add console.log with the text we want logged in the console in parentheses
