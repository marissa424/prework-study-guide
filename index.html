CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • Box shadow adds shadow effects around an elements frame
  • Declorations contain two important components. The css propert we want to apply and the value of the property
  • Linking your style sheet: link rel="stylesheet" href="./assets/style.css"

Git

  • Git: Version control software
  • GitHub: Cloud based service that host respositories

      • Pushing Changes to Git

    1. Make sure you're in the correct branch, run command: cd prework-study-guide
    2. Make sure changes go to branch not main, run command: git checkout -b feature/startercode
    3. Save changes in VScode then start commit to changes, run command: git add -A
    4. Commit Message, run command: git commit -m "enter reason for change here"
    5. Next run command: git pull origin main
    6. Next run command: git push origin feature/starter-code

  • Make sure to always push to your branch and not main
  • Make sure when checking pull request that base: main and compare: feature/starter-code are selected
